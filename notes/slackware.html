Installer Slackware

26 septembre 2014

Contrairement à ce que j'imaginais, Slackware s'installe très facilement. Il faut juste prendre le temps de lire ce que l'installateur dit et... lire la doc !

Je n'ai pas installé les packages contenus dans kde/ et kdei/. J'ai installé tout le reste. J'obtiens donc une slackware 14.1 avec XFCE. Le gestionnaire d'affichage par défaut est XDM. Pour être honnête, j'ai trouvé ca moche. Il faudra le changer.

Pour arriver à mon système actuel, j'ai utilisé tout ou partie des pages de documentation suivantes :

  • la FAQ francophone.
  • la documentation d'installation officielle ;
  • la documentation de post-installation francophone, sauf pour la partie mise à jour du système que je fais avec slackpkg ;
  • la documentation pour franciser slackware, sauf pour la disposition du clavier où j'ai utilisé la doc officielle, plus à jour ;

Suite à pas mal de problème de connexion avec le mirroir OVH, j'ai configuré le mirroir de l'université de Berlin pour slackpgk.

Deux liens à garder en référence :

  • La documentation officielle.
  • Le wiki de la communauté française.
