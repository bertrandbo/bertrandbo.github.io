Installer SLiM sur Slackware

27 septembre 2014

Pour remplacer XDM, j'ai choisi SLiM (utilisé notamment, par Crunchbang). La documentation francophone de slackware couvre l'essentiel pour installer et configurer SLiM.

Je n'ai juste pas fait l'installation à la main comme indiqué dans la documentation (i.e. make install). J'ai préféré utiliser le SlackBuild disponible pour ma version.

J'utilise le thème SLiM de ngc891.

Points d'attention

Par défaut, slackware est configuré en clavier us. SLiM sera donc en qwerty. Il faut penser à changer la disposition du clavier par défaut du système.

Lors du login, obtenir le message failed to execute login command signifie que le fichier .xinitrc n'existe pas dans le répertoire utilisateur. Dans ce cas, copier celui du compte root.
